### Sobre o projeto
- O shopping_cart surgiu da vontade de práticar flutter, aprimorar conceitos e de construir uma e-commmerce
- inspirada no varejo. Nesse projeto foram práticados:
- Requisição Api Rest
- Testes Unitários
- Testes de Widget
- Princípios SOLID
- Gerenciamento de Estado
- HooksRiverpod
